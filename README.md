I'm **Nipurn Agarwal**, founder of **[100XPrompt](https://100xprompt.com)** — an AI coding-agent stack: a terminal-native CLI, an SDK for building your own agents, and a plugin platform, together doing **9,000+ npm installs a month**. Around the core product I build the infrastructure that agentic coding needs — **chitta**, a local memory layer that cuts context tokens up to **143×**; **MCP servers** used with Claude Code, Cursor & Codex; and research into LLM inference efficiency (**turboquant** — 5–6× KV-cache compression). I also work across **voice AI** and **LLM infrastructure**, mostly in TypeScript, Python and Go.

## 🚀 What I'm Building

**[100XPrompt](https://100xprompt.com)** — an AI coding agent that plans, edits, reviews and ships code from your terminal. Multi-model, with hooks, skills, sub-agents and MCP support.

<a href="https://www.npmjs.com/package/100xprompt-cli"><img src="https://img.shields.io/npm/v/100xprompt-cli?style=flat-square&logo=npm&label=100xprompt-cli&color=CB3837"/></a>
<a href="https://www.npmjs.com/package/@100xprompt/sdk"><img src="https://img.shields.io/npm/v/@100xprompt/sdk?style=flat-square&logo=npm&label=@100xprompt/sdk&color=CB3837"/></a>
<a href="https://www.npmjs.com/package/@100xprompt/plugin"><img src="https://img.shields.io/npm/v/@100xprompt/plugin?style=flat-square&logo=npm&label=@100xprompt/plugin&color=CB3837"/></a>

- 🧠 **chitta** — permission-aware, permanent local memory for AI coding agents. Remembers across sessions, cuts context tokens **up to 143×**, and spends **zero LLM tokens** to store or recall.
- 🔬 **turboquant** — 5–6× KV-cache compression for LLMs with minimal accuracy loss.
- 🔌 **[duckduckgo-mcp](https://github.com/Nipurn123/duckduckgo-mcp)** — free, unlimited web-search MCP server. No API keys. Search, crawl & research with AI-ranked results.
- 📄 **[ilovepdf-mcp](https://github.com/Nipurn123/ilovepdf-mcp)** — MCP server for the iLovePDF / iLoveIMG API. 20+ tools: compress, merge, split, convert, watermark, OCR.
- 🗣️ **Voice AI** — real-time voice assistants and Indic speech model fine-tuning (Moshi / LiveKit).
